Description
========
JLink_ARM-OB 是一個採用 STM32 的 JLink 燒錄器，為了縮減電路體積，僅有 [SWD](http://en.wikipedia.org/wiki/Joint_Test_Action_Group#Serial_Wire_Debug) ( Serial Wire Debug ) 功能，除了上述與無法檢測電壓外，其餘功能與 JLink 一樣。
